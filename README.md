# NavYatra: Know Your India

> **An AI-Powered Travel Recommendation System to Explore the Incredible Diversity of India**

Know Your India is a modern web application that helps users discover the best travel destinations across India based on their preferences, budget, travel style, and interests. The platform uses an intelligent recommendation engine to provide personalized travel suggestions while educating users about India's rich culture, heritage, festivals, cuisine, and tourism.

# 🧠 How It Works

The recommendation engine asks users questions such as:

- Budget
- Season
- Travel Purpose
- Travel Duration
- Mountains or Beaches
- Adventure or Relaxation
- Solo, Family, Friends or Honeymoon
- Preferred Climate

Each answer contributes points to matching destinations.

The system calculates scores and recommends the best travel destinations with personalized match percentages.
